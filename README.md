# 歡迎使用 Han Count !

這是紀錄某新聞台提到有關`"韓"`的次數的紀錄程式，由nodejs + gcp編寫而成。

身為高雄北漂工程師，響應韓總發財政策，決定用程式碼協助追隨韓總腳步。

以下網站會追蹤韓總新聞台提到韓總的次數，並即時顯示在畫面上。

[韓總計數器](http://35.199.161.19/)

## 如何開始

1. 把 `_env.js` 更改成 `env.js`。

2. 更改`env.js`設定

3. 輸入 `npm install`

4. 運行 `node .`，如果執行失敗再`npm install`一次。
