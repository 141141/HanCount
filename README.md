# 歡迎使用 Han Count !

這是紀錄某新聞台提到有關`"韓"`的次數的紀錄程式，由nodejs + gcp編寫而成。

身為高雄北漂工程師，響應韓總發財政策，決定用程式碼協助追隨韓總腳步。

## 如何開始

1. clone專案

2. 把 `_env.js` 更改成 `env.js`。

3. 更改`env.js`設定

4. 開啟終端機，輸入 `npm install`

5. 運行 `node .`，如果執行失敗再`npm install`一次。

6. 點擊log網址開啟網站。
